# Enable Google Calendar API

To make the calendar sync work, you need to enable the Google Calendar API in your Google Cloud project.

## Quick Steps:

1. Go to: https://console.cloud.google.com/apis/library/calendar-json.googleapis.com?project=1098786978429
   - Or navigate: Google Cloud Console → APIs & Services → Library → Search "Google Calendar API" → Click on it → Enable

2. Click the **"Enable"** button

3. Wait a few seconds for it to propagate

4. That's it! The sync should work now.

## Alternative Method:

1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Select your project (project ID: 1098786978429)
3. Click **"APIs & Services"** → **"Library"**
4. Search for **"Google Calendar API"**
5. Click on **"Google Calendar API"**
6. Click **"Enable"**

## Verify It's Enabled:

After enabling, you should see:
- Status: "API enabled"
- A green checkmark or "Enabled" badge

Once enabled, the calendar sync feature will work!
